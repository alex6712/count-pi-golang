# Count Pi Golang

Параллельная реализация вычисления числа π с использованием ряда Лейбница на Go.

## 📌 Описание

Проект демонстрирует простой способ приближённого вычисления числа π через ряд Лейбница:

π = 4 * (1 - 1/3 + 1/5 - 1/7 + ...)

Вычисление распараллелено с использованием goroutines и `sync.WaitGroup`, что позволяет эффективно задействовать несколько CPU-ядер.

## ⚙️ Особенности

* Параллельное вычисление с использованием goroutines
* Автоматическое определение количества воркеров (`GOMAXPROCS`)
* Деление задачи на чанки
* Подсчёт времени выполнения
* Сравнение с эталонным значением π (`math.Pi`)

## 🚀 Запуск

Можно запустить файл напрямую:

```bash
go run main.go
```

А также скомпилировать его в исполняемый файл:

- Linux / MacOS

```bash
go build -o main main.go

./main
```

- Windows

```powershell
go build -o main.exe main.go

main.exe
```

## 🔧 Параметры

Можно задать количество итераций через флаг:

```bash
go run main.go -it=50_000_000
```

| Флаг       | Описание                          |
| ---------- | --------------------------------- |
| `-it/--it` | Количество итераций ряда Лейбница |

По умолчанию используется:

```go
const ITERATIONS = 100_000_000
```

```

Эталон π:     3.1415926535897931
Погрешность:  -0.0000000000115956
Время работы: 16.801 сек
```

## 🧠 Как это работает

1. Диапазон итераций делится на равные части (чанки)
2. Каждый воркер вычисляет свою часть суммы
3. Результаты собираются в общий массив
4. Итоговая сумма умножается на 4

## ⚠️ Замечания

* Ряд Лейбница сходится **очень медленно**, поэтому для высокой точности требуется большое количество итераций
* Производительность сильно зависит от количества доступных CPU-ядер
* Это демонстрационный пример, а не оптимальный способ вычисления π
